class  Adafruit_NeoPXL8
 Adafruit_NeoPXL8 is a subclass of Adafruit_NeoPixel containing buffers for DMA-formatted 8-way concurrent output. Once a transfer is initiated, the original NeoPixel data can then be modified for the next frame while the transfer operates in the background. More...

 Adafruit_NeoPXL8HDR is a subclass of Adafruit_NeoPXL8 with additions for 16-bits-per-channel color, temporal dithering, frame blending and gamma correction. This requires inordinate RAM, and the frequent need for refreshing makes it best suited for multi-core chips (e.g. RP2040). More...
 

Detailed Description

8-way concurrent DMA NeoPixel library for SAMD21, SAMD51, RP2040, and ESP32S3 microcontrollers.
